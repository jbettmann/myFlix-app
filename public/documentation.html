myFlix API

This web application provides users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL Request Parameter HTTP Method Request body data format Response body data format
Returns a list of all movies to the user "/movies" none GET none A JSON object with data on list of all the movies
Example: {Title: "exampleTitle", Description: "movie description here", Director: { Name: "directorName", Bio: "bio description" }, }
Returns data about a single movie by title to the user "/movies/:title" :title GET none A JSON object with data about a single movie
Example: Example: {Title: "exampleTitle", Description: "movie description here", Director: { Name: "directorName", Bio: "bio description" }, }
Returns data about a genre by name "/movies/genre/:genreName" :genreName GET none A JSON object with data about a genre
Example: Genre: { "name": "Thriller", "description": "description..."}
Returns movies an actor is in by actor's name "/movies/actors/:actorName" :actorName GET none Example: Example: {Title: "exampleTitle", Description: "movie description here", Director: { Name: "directorName", Bio: "bio description" }, }
Returns details about a director by name "/movies/directors/:directorName" :directorName GET none A JSON object with details about a director
Exmple: { Name: "directorName", Bio: "director bio", Age: "56" },
Allows new users to register "/users"/ none POST JSON object with data about the user to add
Example: { Username: "exampleName", Password: "password1", Email: "email@gmail.com"} 		JSON object with data about added new user
Example: { ID: 5, Username: "exampleName", Password: "*********", "favoriteMovies":[] },
Allows users to update their info "/users/:id" :id PUT none A text message indicating the user info was updated
Allows users to add a movie to their Favorites list "/users/:id/favorites/:movieTitle" :movieTitle POST none A text message indicating that a movie has been added to their Favorites list.
Allows users to remove a movie from their Favorites list "/users/:id/favorites/:movieTitle" :movieTitle DELETE none A text message indicating that a movie has been deleted from their Favorites list.
Allows users to add a movie to their To Watch list "/users/:id/toWatch/:movieTitle" :movieTitle POST none A text message indicating that a movie has been added to their To Watch list.
Allows users to remove a movie from their To Watch list "/users/:id/toWatch/:movieTitle" :movieTitle DELETE none A text message indicating that a movie has been deleted from their To Watch list.
Allows existing users to deregister "/users/:id" :id DELETE none A text message indicating that the user has been deleted.
