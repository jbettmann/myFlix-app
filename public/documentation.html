myFlix API

This web application provides users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL Request Parameter HTTP Method Request body data format Response body data format
Returns a list of all movies to the user "/movies" none GET none A JSON object with a list of all the movies
Example:
{
Title: "exampleTitle",
Description: "movie description...",
Director: {
Name: "directorName",
Bio: "bio description"...
},...
}
Returns data about a single movie by title to the user "/movies/:title" :title GET none A JSON object with data about a single movie
Example:
{
Title: "exampleTitle",
Description: "movie description...",
Director: {
Name: "directorName",
Bio: "bio description"...
},...
}
Returns data about a genre by name "/movies/genres/:genre" :genre GET none A JSON object with data about a genre
Example: {
"Comedy is a genre of film in which the main emphasis is on humor. These films..."
}
Returns movies an actor is in by actor's name "/movies/actors/:actor" :actor GET none A JSON object with data about a single movie
Example:
{
Title: "exampleTitle",
Description: "movie description...",
Director: {
Name: "directorName",
Bio: "bio description"...
},...
}
Returns details about a director by name "/movies/directors/:director" :director GET none A JSON object with details about a director
Exmple:
{
"Name": "Wes Anderson",
"Bio": "UPDATE: His new-age style is one to remember. Focused on cinematography and dialog!",
"Birth": "1969-05-01T00:00:00.000Z"
}
